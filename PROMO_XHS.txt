A-SOUL 粉丝应援工具 — 一键签到 + 视频点赞 + 动态点赞

给嘉然、贝拉、乃琳、心宜、思诺的直播间签到、视频点赞、动态点赞，设置一次，每天自动跑。

不需要装任何软件，不需要会编程，只要有 GitHub 账号就行。

——————————————

📌 支持的功能

✅ 自动佩戴粉丝牌 + 直播间弹幕签到（5个直播间全签）
✅ 自动给成员新视频点赞
✅ 自动给成员新动态（图文/转发）点赞
✅ 投币/收藏（可选，默认关闭不消耗硬币）

——————————————

📦 设置步骤（只需一次，约5分钟）

① Fork 仓库
打开 github.com/XiaoYiWeio/asoul-support
点右上角「Fork」按钮，复制到你自己账号下

② 获取 B站 Cookie
在 Chrome 打开 bilibili.com（确保已登录）
按 F12 → Application → Cookies → bilibili.com
找到 SESSDATA 和 bili_jct，复制它们的值

③ 填入 GitHub Secrets
进入你 Fork 的仓库 → Settings → Secrets and variables → Actions
点 New repository secret，添加两个：
  - Name: SESSDATA    Value: 你的SESSDATA
  - Name: BILI_JCT    Value: 你的bili_jct

④ 启用 Actions
进入仓库 Actions 标签 → 点「I understand my workflows, go ahead and enable them」

搞定！每天北京时间 10:30 自动执行。

——————————————

💡 不想等定时？手动跑一次

Actions 标签 → 左侧选「A-SOUL 每日应援」→ 点 Run workflow → Run

——————————————

🔒 安全说明

你的 Cookie 只存在你自己的 GitHub 里，加密存储
任何人（包括我）都看不到你的 Cookie
所有代码完全开源，可以自己检查
只做签到和点赞，不会改你的账号设置
