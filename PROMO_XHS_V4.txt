一个魂们！你的粉丝牌亲密度还在手动挂机涨吗？

我把之前那个 A-SOUL 自动签到工具大升级了（v4.0），这次是真·能涨亲密度了：

【核心功能】

成员开播后全自动：
发弹幕点亮牌子 → 心跳挂机涨亲密度 → 下播自动停

实测效果：30/30 亲密度直接拉满，粉丝牌当场升级

每个成员每天 30 亲密度上限，5 分钟涨 6 点，挂 25 分钟就满了。你甚至不需要打开 B站。

----

【一键安装，对话配置】

OpenClaw 用户安装只需一行命令：

npx clawhub install asoul-support

装完之后告诉你的 OpenClaw：
"帮我设置 A-SOUL 自动挂机"

它会引导你填 B站 Cookie，然后自动帮你配好定时任务。
不需要看文档，不需要学命令，全程跟 AI 对话就搞定。

----

【没有 OpenClaw 也能用】

不用 OpenClaw 的话，也可以 Fork GitHub 仓库，自动帮你视频点赞和动态点赞（但不能涨亲密度，涨亲密度需要 OpenClaw）。

----

【和之前版本有什么不一样】

v3.0：心跳协议过期了，看起来成功实际上不涨亲密度（B站偷偷改了接口）
v4.0：换成了最新的移动端心跳协议，纯 Python 实现

简单说就是：以前的版本白挂了，现在这个是真挂真涨

----

【完整功能】

自动的：
- 每 30 分钟检测一次成员是否开播
- 开播了自动发 10 条弹幕点亮粉丝牌
- 自动心跳挂机涨亲密度（每天 30 满额）
- 下播了自动停止并通知你
- 每 2 天自动给新视频点赞

----

【技术亮点】

纯 Python 实现，零外部依赖
不需要 Node.js，不需要 Docker，不需要额外服务
一个 Python 文件搞定一切

----

【安全说明】

Cookie 加密存储，开源代码可自查
只做点赞和弹幕，不改账号设置
不投币不送礼不关注陌生人
