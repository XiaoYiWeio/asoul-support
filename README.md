# A-SOUL Support

A-SOUL 粉丝自动应援工具 — 直播间签到 + 视频互动。

## Features

- **直播间签到** — 一键给嘉然、贝拉、乃琳、心宜、思诺的直播间发弹幕签到
- **视频点赞** — 批量给成员新视频点赞
- **视频投币/收藏** — 可选的投币和收藏功能

## 使用方式

通过 OpenClaw 对话：

```
请给asoul直播间都签到一下
请给asoul 3月份新发布的视频都做一下点赞
给asoul最近的视频三连（点赞+投币+收藏）
```

## 前置条件

需要 B站 Cookie（SESSDATA + bili_jct）。如已在 `bilibili-live-checkin` 中设置，会自动复用。

## 零依赖

全部使用 Python 标准库，无需安装任何包。
